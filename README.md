## Установка

```bash
bun install
```

## ИИ разработка

Для упрощения ИИ разработки есть настроенные правила (https://github.com/intellectronica/ruler). С помощью этих правил можно сгенерировать правила для любых агентов

```bash
bun ruler:apply --agents claude
```

Для установки новых скиллов используется

```bash
npx skills add https://github.com/angular/skills
```
